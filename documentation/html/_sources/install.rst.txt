
############
Installation
############

Dependencies:
-------------

Python 3.10

pygame

Install:
--------

Install pygame by running the following command::

  python3 -m pip install -U pygame --user

Clone Repo::

  git clone https://github.com/marinome/EECS448_Project3.git


At the command line, on branch ``Project4``::

  python3 snakegame.py
