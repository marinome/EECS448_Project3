Snake Game Module¶

snakegame.chooseDifficulty(snake)¶

Option after starting to select difficulty, changes things like game speed/speed over time

Author: Michael Talaga

Parameters

snake (Snake) – Snake object

snakegame.game(snake, difficulty)¶

Main game loop to play snake game

Author: Michael Talaga

Parameters

  • snake (Snake, made of, Block) – This is the snake object which will be moving around on the screen. The user will be able to move this with arrow keys or wasd.

  • crash (Pygame mixer using mp3 file) – Crash sound upon impact

snakegame.game_over(font_type, font_size, font_color, final_score)¶

display final score over board

Parameters

  • font_type (pygame font type) – font style

  • font_size (int) – font size

  • font_color (RGB color) – font color

  • final_score (int) – final score of game to display

snakegame.get_color(color_string)¶

pass name of color, return RGB value for color

Parameters

color_string (String) – name of color

Return _

The RGB value of the passed color

Return type

Tuple (red, green, blue)

snakegame.gridScreen(gridSize, color1, color2)¶

this will add grid lines to the screen

Author: Mason Otto

Parameters

  • color1 (RGB color, (R,G,B), 0-255 each) – grid color 1

  • color2 (RGB color, (R,G,B), 0-255 each) – grid color 2

snakegame.main()¶

Main file to run the game and exit the system when it is finished.

Author: Michael Talaga, AMA

snakegame.menu()¶

Menu to start the game or quit

Author: Michael Talaga

Returns

True if the game will start or False to quit.

Return type

Boolean

snakegame.show_score(score_2_display, color, font, size)¶

displays score on screen

Parameters

  • score_2_display (int) – score that will be displayed

  • color (RGB color) – color of the score font

  • font (pygame font name) – font style

  • size (int) – font size
