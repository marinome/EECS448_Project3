Snake Module¶

class snake.Snake(x, y)¶

Bases: pygame.sprite.Sprite

Creates a snake class which will move around on the board and be controled by the user. If it encounters food it will grow the snake. If the snake touches the edge of the board, the game will end.

Example:

snake =  Snake(100, 200)
snake._(_)
Parameters

  • size (int) – Size of snake

  • head (Block) – The head of the snake

  • tail (Block) – tail of the snake

  • blocks (sprite group from pygame's library which) – A collection of blocks for each block of the snake

  • direction (String) – The direction the snake is currently moving - defaulted to still but can be up,down,left, or right

update(screenDimensions, food)¶

Update movement of snake object via keyboard pressing.

Author: Michael Talaga

Parameters

  • food (Food) – Food object to check for collision

  • change (Boolean) – Dictating if the snake will change its direction. Blocks will readjust location based on this.

  • y_change (int) – The amount the snake moves in the vertical direction per turn (can be positive or negative)

  • x_change (int) – The amount the snake moves in the horizontal direction per turn (can be positive or negative)
